## DEFINIR ESTILOS DE MARCADO

- ` **Negrita** `
- ` *Cursiva* `
- ``` `Strings` ```
- ` ``` Código Multi-Línea``` `
- ` ~ Tachado ~ `
- `*`, `-`, `+` Listas Desordenadas.
- `1.`, `I.`, `i.`, `A.`, `a.` Listas Ordenadas.
- `- [ ] Check-Box (vacía)`
- `- [x] Check-Box (marcada)`
- Encabezados :
	- `#`, `##`, `###` ... `#######`
- Citas :
	- `>`
	- `> > >`
